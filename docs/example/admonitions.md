# 警告框

英文名"Admonitions"，中文可叫做"警告框"、"高亮块"等。现将所有支持的类型及其参数罗列如下，以备不时之需。

/// note

`note`, `seealso`

///

/// seealso

`seealso`

///

/// abstract

`abstract`, `summary`, `tldr`

///

/// info

`info`, `todo`

///

/// tip

`tip`, `hint`, `important`

///

/// success

`success`, `check`, `done`

///

/// question

`question`, `help`, `faq`

///

/// warning

`warning`, `caution`, `attention`

///

/// failure

`failure`, `fail`, `missing`

///

/// danger

`danger`, `error`

///

/// bug

`bug`

///

/// example

`example`

///

/// quote

`quote`, `cite`

///
