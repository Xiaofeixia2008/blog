## 问题提出：

1. 健康饮品种类较为单一，难以满足学生多元化的消费需求。

  目前小卖部提供的饮品多以传统碳酸饮料及含糖茶饮为主，缺乏符合《校园商超规范管理指导意见（试行）》导向的 NFC 鲜榨果汁、豆奶制品及无糖茶饮等健康品类，未能充分兼顾学生的口味选择与膳食健康需求。

2. 小卖部内部空间规划欠佳，导致高峰期排队拥堵，影响购买效率。

  由于杂物堆放挤占了排队空间，现有排队动线极易与货架区域形成干扰，不仅降低了课间及午间的结账效率，也给学生密集时段的通行与购物带来了不便。

3. 早餐及课间时段的面包供应不稳定，品类缺乏定期更新。

  新鲜面包作为学生早餐及课间补充能量的重要来源，目前存在供应量不足、品类长期单一的问题，未能根据学生口味变化进行动态调整，影响了学生的日常饮食体验。

4. 校园内缺乏新鲜水果的便捷购买渠道。

  当前小卖部尚未提供盒装果切或整果售卖服务，学生在早、中、晚时段难以就近补充新鲜水果，不利于均衡膳食结构的养成。

5. 售卖终端硬件设施不足，高峰时段承载压力大。

  仅依赖单一的小卖部实体窗口进行销售，在人流高峰期己显吃力。增设具备校园一卡通支付功能的自动贩卖机，可作为分流人流、缓解排队压力的有效措施，但该项基础设施目前尚处于空白状态。

## 情况分析：

（图片 x 3）

通过在校园内展开问卷调查，我们总共收集到 232 份有效问卷。其中，84.91% 的同学在小卖部的主要消费类别是饮品类；而当问及对目前小卖部饮品选择的满意程度时，较多同学指出可以进行改进。72.41% 的同学希望可以引进一些其他的无糖或低糖饮料，如果汁等，以增加饮品选择；而对于零食类，许多同学建议可以增加如面包蛋糕的类别，增设新鲜水果的供应等。经调查反馈，一些同学中午可能因为社团或学生工作等原因无法按时到饭堂就餐，中午只能挨饿。因此建议在小卖部货架上售卖余量的面包，蛋糕等（因为现在大部分面包和蛋糕在上午即售完了），巧克力等能短暂补充能量的食品。

同时，在问卷末尾的反馈中，有同学反映小卖部空间较小，导致排队时长过长和拥挤问题，排队队伍挡住货柜，很不方便。

## 具体建议：

针对以上问题，为改善同学们的购买体验，提高学校生活质量，满足同学们的合理需求，我们现提出以下建议。

1. 丰富小卖部售卖饮品种类

  根据广东省《校园商超规范管理指导意见（试行）》和现行法规制度，我们现对下列几个种类提出建议，

  1. NFC / FC 100% 新鲜果汁
  2. 豆奶、谷粒多
  3. AD 钙奶、养乐多
  4. 东方树叶，三得利无糖乌龙茶等低糖饮品

  增加饮品种类可以丰富选择，满足同学们对于高品质生活的需求，且大都为健康选项，确保不会影响同学们的饮食安全。

2. 优化小卖部内部布局

  小卖部内长期堆放箱子等杂物，导致排队队伍布局不理想，排队经常会阻碍货柜。建议将箱子等物品挪开，空间用于排队，提高结账效率。

3. 增加售卖的新鲜面包种类

  在早餐和大课间时段增加面包的供应量，并定期更换新品类。

4. 增加售卖新鲜水果

  可以通过盒装果切、整果售卖等形式售卖。便于同学们在大课间和中午饭后补充新鲜水果。

5. 增设自动贩卖机

  自动贩卖机可联通校园一卡通系统，使同学们能更快更方便地购买到饮品，解决小卖部空间不足的问题，分散小卖部人流，腾出位置售卖新增商品。且平均造价在 8000 到 1 万之间，价格较为实惠。
